# 주요 기능
⭐ 위치기반 검색
- 사용자가 원하는 위치를 검색하면 해당 위치 근거리에 있는 관광명소 목록을 제공

⭐ 기록 기능
- 사용자가 하고자 하는 여행계획을 좀 더 원활하게 계획할 수 있도록 제공

⭐ 최적의 교통편 제공
- 사용자의 출발위치와 목적지를 기반으로 최적의 교통편을 제공

⭐ 유저들의 상호작용
- 해당 목적지의 플래너 공유로 추천 및 게시글/댓글 및 추천하는 서비스를 구현해서 유저들 간의 소통 서비스 구현
